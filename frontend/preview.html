📚 University Library Management System

Modern Web Interface with Beautiful Dashboard

🏠 Interactive Dashboard

Real-time statistics, recent activity feed, and visual metrics with beautiful gradients and animations.

📚 Book Management

Add, view, search, and manage books with sortable tables and intuitive modal forms.

👥 User Management

Complete user registration, profile management, and borrowing history tracking.

🔄 Borrow/Return System

Streamlined borrowing and return operations with real-time status updates.

🧭 Shelf Navigation

Visual pathfinding between library shelves using advanced graph algorithms.

Personalized book suggestions using dynamic programming and edit distance algorithms.
